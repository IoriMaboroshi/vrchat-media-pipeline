VRChat BPlayer Proxy

控制台 统计 生成器 预加载 API 文档 设置 退出

概述

VRChat BPlayer Proxy 是一个 B 站视频实时代理服务。它通过服务器端 FFmpeg 将 B 站的 DASH 分离流（音视频独立）实时封装为单一 MP4/M3U8 流，使 VRChat 等播放器可以直接播放高清视频。

提示：所有 API 请求都需要携带 token 参数。请妥善保管你的 Token，不要泄露给他人。

认证

所有需要认证的 API 端点都需要在 URL 参数中携带 Token：

?token=thechwinlyu

Token 错误或缺失将返回 403 Forbidden。

以下端点无需 Token：/health

GET /play — 获取视频流 核心端点

返回实时封装的视频流，可直接在 VRChat/PotPlayer/浏览器中使用。

请求参数

参数类型必填说明
bvidstring是*B 站视频 BV 号，如 BV1GJ411x7h7
ep_idstring是*番剧/电影 ep_id（与 bvid 二选一），如 ep1482617
tokenstring是API Token
qxstring否清晰度，默认 1080p
cidint否直接指定分P cid
pageint否分P编号 (1-indexed)
tint否起始时间(秒)，用于视频跳转
rawint否设为 1 强制返回 m3u8 跳转（用于播放器无法识别时）

qx 清晰度参数

qx 值对应画质B站 qn要求
4k4K 超清120需大会员
1080p601080P 60fps112需大会员
1080p1080P 高清64登录即可
720p720P 高清32—
480p480P 清晰16—
360p360P 流畅8—

若视频本身不支持请求的清晰度，API 会按最佳→最低顺序自动回退。

示例

默认 1080P：

http://your-server:14515/play?bvid=BV1GJ411x7h7&token=your_token

1080P60 高画质：

http://your-server:14515/play?bvid=BV1GJ411x7h7&token=your_token&qx=1080p60

从 30 秒处开始播放：

http://your-server:14515/play?bvid=BV1GJ411x7h7&token=your_token&t=30

番剧播放：

http://your-server:14515/play?ep_id=ep1482617&token=your_token

响应头

头名称说明
X-Content-Duration视频时长（秒）
X-Content-Title视频标题（URL 编码）
X-Stream-Id流 ID，用于查询编码状态

POST /api/preload — 预加载视频 新增

预加载视频到本地缓存，后续播放将直接使用本地文件，无需从 CDN 重复下载。缓存有效期12小时。

请求参数

参数类型必填说明
bvidstring是B 站视频 BV 号（JSON body）
qxstring否清晰度，默认 1080p（JSON body）
tokenstring是API Token（URL 参数）

示例

POST http://your-server:14515/api/preload?token=your_token
Content-Type: application/json

{"bvid": "BV1GJ411x7h7", "qx": "1080p"}

GET /api/cache-stats — 缓存状态 新增

查看当前 DASH 缓存状态，包括已缓存视频的标题、封面、时长、大小、过期时间等。

请求参数

参数类型必填说明
tokenstring是API Token（URL 参数）

示例

http://your-server:14515/api/cache-stats?token=your_token

DELETE /api/cache/{key} — 删除缓存 新增

删除指定的缓存条目。key 可通过 cache-stats 接口获取。

请求参数

参数类型必填说明
keystring是缓存条目 key（URL 路径）
tokenstring是API Token（URL 参数）

示例

DELETE http://your-server:14515/api/cache/abc123def456?token=your_token

GET /api/aria2-settings — 下载线程设置 新增

查看当前 aria2 下载线程数配置。

示例

GET /api/aria2-settings

返回

{"connections": 32}

POST /api/aria2-settings — 修改下载线程 新增

修改 aria2 下载线程数（1-64）。线程越多下载越快，但会增加网络负载。

请求参数

参数类型必填说明
connectionsint是线程数，范围 1-64（JSON body）
tokenstring是API Token（URL 参数）

示例

POST /api/aria2-settings?token=your_token
Content-Type: application/json

{"connections": 32}

GET /api/token-info — Token 信息 新增

获取当前 API Token 信息（masked 版本），用于 URL 生成器。

示例

GET /api/token-info

返回

{"token": "thechwinlyu", "masked": "thec****lyu"}

GET /api/stream-status/{stream_id} — 流状态 新增

获取 MP4/M3U8 编码进度。用于 Web 面板显示进度条。

返回

{
  "stream_id": "abc12345",
  "status": "processing",
  "progress_pct": 45.2,
  "eta_seconds": 30
}
status 值说明
processing正在编码
completed编码完成
not_found流不存在或已过期

错误码

HTTP 状态码含义
200成功
206部分内容（Range 请求）
403Token 缺失或错误
400参数错误（如 BVID 无效、视频不可用）
503未登录 B 站 或 服务器繁忙（并发超限）
